Exibir saudação para usuário após login.

Para fazer com que os usuário se sintam mais a vontade o sistema deve exibir uma mensagem de saudação logo após entrar no sistema.

O sistema exibe uma mensagem de saudação com o nome do usuário e com concordância nominal correta.

Exemplos

Nome Sexo Saudação
Jorge M Bem-vindo, Jorge.
Maria F Bem-vinda, Maria.
